# goftp
轻松共享你的目录


# 启动方法
```
1. linux下的启动方法
./ftp  /data  2121    #第1个参数是你要共享的目录 ，第二个参数是自定义端口

2. windows下的启动方法
ftp.exe  c:  2121      #此时你的c盘已经被共享

```


# 用法
客户端从浏览器访问：  http://x.x.x.x:2121/     
x.x.x.x 代表启动ftp的服务器ip
